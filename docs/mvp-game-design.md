# Structure MVP

## First Playable Game
Tower Builder

### Mission
Build a 200 ft communications tower.

Constraints:
- Wind load: 20 psf
- Drift limit: H/200
- Lowest cost wins

### User Actions
- Place node
- Draw member
- Delete member
- Analyze

### Member Types
Small
Medium
Large

### Visual Style
- Blueprint background
- Retro-inspired engineering aesthetic
- Compression members shown in red
- Tension members shown in blue
- Force magnitude shown by line thickness

### MVP Architecture
Frontend Only
- React
- TypeScript
- Vite
- SVG canvas
