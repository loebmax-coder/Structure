# Structure Roadmap

## Version 0.1

### Goal
Create a browser-based structural topology optimization tool.

### Frontend
- React
- TypeScript
- Vite
- Geometry editor
- Load editor
- Support editor

### Backend
- FastAPI
- NumPy
- SciPy
- CVXPy

### Features
- Point loads
- Pin supports
- Roller supports
- Fixed supports
- Required geometry regions
- Forbidden geometry regions
- Truss topology optimization
- Reactions
- Member forces
- Deflections

## Future Vision

- Frame optimization
- Arch generation
- Cable-supported systems
- Cost optimization
- Carbon optimization
- DXF export
- Engineering reports
- AI-assisted concept generation
